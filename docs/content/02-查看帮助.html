2-查看帮助

查看帮助

$ make help
Usage:
  image          构建镜像
  container      构建容器
  html           生成静态文件
  pdf            生成 PDF
  serve          启动本地 web 服务，监听 4000 端口
  exec           生成可执行文件（需要使用 Golang 编译）
  clean          删除命令 html pdf serve 生成的中间物
  rm-container   删除容器
  rm-image       删除镜像
  rm             删除容器和镜像
  help           打印命令帮助信息
